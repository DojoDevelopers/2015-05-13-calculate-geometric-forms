# Dojo Calculate Geometric Forms

## Desafio

Dado as informações deve ser calculado a área das figuras geométricas :

##### Triângulo
```
b = 10
h = 4
s  = área deve ser 20

S = (b.h) / 2
```

##### Paralelograma
```
b = 5.2
h = 1.5
s  = área deve ser 7.8

S = b.h
```

##### Losango (Paralelograma especial) 
```
b = 6
h = 6
s  = área deve ser 18

S = (b.h) / 2
```

##### Trapézio
```
B = 8
b = 14
h = 6
s  = área deve ser 66

S = (B+b).h / 2
```

##### Quadrado
```
L = 8
s  = área deve ser 64

S = 8²
```
